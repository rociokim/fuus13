# CLI 명령어

* pwd : 현재 디렉토리 확인하기(디렉토리 : 폴더)
* cd : 디렉토리 이동하기
* cd .. : 상위 디렉토리 이동
* ls : 리스트 보기
* cat : 파일 내용 확인
* rm-rf : 코스 삭제
* mkdir : 디렉토리 생성하기
* touch : 파일 생성
* echo : 파일 생성(내용까지 추가 가능)
* mv : 이름변경 및 이동
* cp : 복사
* rmdir : 폴더제거(비어있는 폴더만 삭제가능)
* rm : 삭제
``` js
mkdir fus13(폴더명)
```
